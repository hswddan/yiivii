# 伊娃交易日志
外汇,黄金,股指,交易员作手回忆录

* 项目标题：
    * 伊娃交易日志，交易员作手回忆录
    * yiivii

* 项目介绍：
    * 养成认真对待每一笔交易的习惯；
    * 以最简洁的方式记录日常交易得失；
    * 让每一笔交易都能成为盈利道路的垫脚石；

* 技术要点：

    * Thinkphp5.1 + Bootstrap v3 + ZUI + jQuery；
    * 无密码登录，邮箱即账户，随机验证码登录；
